# Norwa Payroll

Kenya Payroll Integration for ERPNext (2025/2026)

## Features
- Salaries Components (NSSF, SHIF, Housing Levy, PAYE)
- Tax Regime Switching (2025 vs 2026 Limits)
- Automatic Formula Injection

## Installation
1. Get App: `bench get-app https://github.com/ITNORWA/Norwa_payroll.git`
2. Install: `bench install-app norwa_payroll`
